This project was completed as part of my **Data Analytics Internship at DecodeLabs**.

After completing **Project 01: Data Cleaning & Preparation**, the next step was to perform **Exploratory Data Analysis (EDA)** on the cleaned dataset.

The purpose of this project was to transform cleaned data into meaningful insights by exploring trends, patterns, customer behavior, and business-related information using Python.

Rather than simply viewing rows and columns of data, EDA helps answer important business questions through statistics and visualizations.

# 📂 Dataset Description

The dataset contains customer order information including:

| Column | Description |
|----------|-------------|
| OrderID | Unique order identifier |
| Date | Order date |
| CustomerID | Customer identifier |
| Product | Product purchased |
| Quantity | Number of items ordered |
| UnitPrice | Price of one item |
| TotalPrice | Total amount paid |
| PaymentMethod | Customer payment method |
| CouponCode | Applied coupon (if any) |
| ReferralSource | Customer acquisition source |
| OrderStatus | Current order status |
| TrackingNumber | Shipping tracking ID |
| ShippingAddress | Customer address |
| ItemsInCart | Number of products in cart |

# 📊 Analysis Performed

## 📈 Descriptive Statistics

The following statistical measures were calculated:

| Measure | Purpose |
|---------|---------|
| Mean | Average value |
| Median | Middle value |
| Count | Total number of records |
| Minimum | Smallest value |
| Maximum | Largest value |
| Standard Deviation | Spread of the data |

---

## 🛍 Product Analysis

The dataset was analyzed to determine:

- Most ordered products
- Product popularity
- Customer purchasing preferences

---

## 💳 Payment Method Analysis

Payment methods were analyzed to understand customer preferences and purchasing behavior.

Examples include:

- Credit Card
- Debit Card
- PayPal
- Cash

---

## 🌍 Referral Source Analysis

Customer acquisition sources were explored to understand which channels contributed the most orders.

Examples include:

- Instagram
- Facebook
- Google
- Email
- Referral

---

## 📦 Order Status Analysis

Order records were analyzed to understand:

- Completed Orders
- Pending Orders
- Cancelled Orders

---

## 📈 Trend Analysis

Trend analysis was performed to identify customer purchasing patterns over time.

This helps understand periods of increased or decreased business activity.

---

## 🚨 Outlier Detection

Outliers were identified using a **Box Plot** to detect unusually high or low values in the **Total Price** column.

Detecting outliers helps identify exceptional transactions that may require further investigation.
