# 📊 Project 02: Exploratory Data Analysis (EDA)

## 📖 About This Project

After cleaning the dataset in Project 01, the next step was to explore and understand the data.

The goal of this project was to discover useful information hidden in the dataset by answering questions such as:

- Which products are ordered the most?
- Which payment methods do customers prefer?
- Are there any unusual or extremely high-value orders?
- What trends and patterns can be observed?

By analyzing the data, I transformed raw numbers into meaningful insights that can support better business decisions.

---

## 🎯 Project Objective

To explore the cleaned dataset, understand customer purchasing behavior, identify trends and outliers, and present the findings through simple statistics and visualizations.

---

## 🛠️ Tools & Technologies

- 🐍 Python
- 🐼 Pandas
- 🔢 NumPy
- 📈 Matplotlib
- 📓 Jupyter Notebook

---

## 📂 Project Files

- **Project_2_EDA.ipynb** – Complete analysis with Python code, charts, and explanations.
- **Cleaned_Dataset.csv** – The cleaned dataset used for analysis.

---

## 📌 What I Explored

During this project, I:

- 📊 Calculated summary statistics such as average values and record counts.
- 📈 Explored customer purchasing trends.
- 💳 Analyzed the most popular payment methods.
- 🛒 Identified the most frequently ordered products.
- 🚨 Detected unusual (outlier) orders.
- 📉 Created charts to make the data easier to understand.
- 💡 Summarized the key insights gained from the analysis.

## ✅ Outcome

This project helped me strengthen my understanding of **Exploratory Data Analysis (EDA)** and demonstrated how data can be transformed into meaningful insights before creating dashboards or making business decisions.
